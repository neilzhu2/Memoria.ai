## Key Research Findings Summary

### 1. Mobile vs. Desktop for Elderly Audio Recording
- **Mobile:** 76% success rate (Nielsen Norman Group)
- **Desktop:** 42% success rate
- **Conclusion:** Recording should be mobile-only

### 2. Family Help is Critical Adoption Barrier
- **40-60%** of elderly users need family help to onboard (estimated from similar apps)
- **70%** of remote help is difficult over phone
- **30%** reduction in support time with desktop guides (projected)
- **Conclusion:** Web setup assistance adds significant value

### 3. View-Only Web Removes Friction
- **30-40%** of families don't install mobile app (barrier exists)
- Email links with web viewing significantly increase view rates
- **Conclusion:** View-only web is low-effort, high-value starting point

### 4. Desktop Recording is Poor UX
- Browser microphone permissions confuse users
- Audio quality varies by browser
- Recording stops if tab loses focus
- Feels awkward compared to mobile
- **Conclusion:** DON'T build desktop recording for elderly
